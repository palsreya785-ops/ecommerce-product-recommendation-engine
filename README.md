# 🛒 E-Commerce Product Recommendation Engine

A Python-based desktop application that recommends products based on user purchase history, search history, cart items, and ratings. This project demonstrates the use of Data Structures & Algorithms (DSA), Object-Oriented Programming (OOP), and GUI development using Tkinter.


## 🚀 Features

- Personalized product recommendations
- Product search
- View all products
- Recommendation ranking
- Report generation
- Interactive Tkinter GUI
- CSV-based data storage

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/palsreya785-ops/ecommerce-product-recommendation-engine.git
```

### Move to the project folder

```bash
cd ecommerce-product-recommendation-engine
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

Windows:

```bash
venv\Scripts\activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```
